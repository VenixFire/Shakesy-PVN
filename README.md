To install and play this video game prototype:

  Download this repo by clicking the Code<> button and clicking "Download ZIP"

  Once you've downloaded this file you can wait to unzip it until you've completed the next steps.

  Install Ren'Py Visual Novel Engine v8.4.1: https://www.renpy.org 

  Once this is installed and opened, unzip the entire folder downloaded from GitHub into your User/Documents/Ren-Py-Projects/ShakesyP folder by right clicking the ZIP file and clicking your computer's option to unzip.
    (You will have to make the ShakesyP folder inside of the Ren-Py-Projects folder)

  From there you can open Ren'Py and play the game using the Launch Project button with the ShakesyP game selected.
