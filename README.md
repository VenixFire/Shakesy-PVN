To install and play this video game prototype:

  Install Ren'Py Visual Novel Engine v8.4.1: https://www.renpy.org 

  Once this is installed and opened, place the entire folder downloaded from GitHub into your User/Documents/Ren-Py-Projects/ShakesyP folder...
    (You will have to make the ShakesyP folder inside of the Ren-Py-Projects folder)

  From there you can open Ren'Py and play the game using the Launch Project button with the ShakesyP game selected.
